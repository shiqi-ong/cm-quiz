# Project Instructions for Codex

This repository hosts a Next.js application for a quiz that generates leads for a B2B website.

## Key Requirements

- **Framework:** Next.js (use TypeScript). The app will be deployed to Vercel.
- **Purpose:** Implement a quiz about cloud modernization. If the user scores 80% or higher, display a message about winning free swag from moderncloud.io and provide a form to collect their contact information.
- **Deployment:** The project will be hosted at <https://github.com/shiqi-ong/cm-quiz> and deployed via Vercel.

## Development Guidelines

- Use functional components and React hooks.
- Keep styling simple; Tailwind CSS is preferred but plain CSS modules are fine.
- Keep the code tidy and document major functions with comments.
- Add tests as the project grows. If `npm test` exists, run it before every commit.
- Update `README.md` whenever a significant feature or setup step is added.
